## Goal

Apply the recent invoice PDF redesign to the **patient profile ‚Üí Invoices tab** so Print, Download, and WhatsApp share all produce the same professional clinic-invoice layout.

## Current state

- `src/lib/invoicePdf.ts` (used by Download and WhatsApp share in `PatientInvoicesTab`) was already rewritten to the new layout ‚Äî clinic logo + name top-left, address/phone/email/website top-right, bordered Invoice/Patient/Doctor section, `Particulars | Unit | Rate | Qty | Amount` table, status badge, double-rule total, footer.
- `src/lib/invoiceUtils.ts ‚Üí buildInvoiceHtml()` (used by the **Print** button via `printInvoice`) still renders the old teal-gradient layout with columns `# | Description | Qty | Rate | Amount`. This is the mismatch.

## Changes

### `src/lib/invoiceUtils.ts`

Rewrite `buildInvoiceHtml(invoice, clinic)` so the printed HTML mirrors the new PDF:

- **Header row** (two columns):
  - Left: clinic logo (if `clinic.logo_url`), clinic name in display font, "TAX INVOICE" / "INVOICE" label.
  - Right: clinic address (multi-line), phone, email, website, GSTIN ‚Äî each on its own line, right-aligned.
- **Meta strip** under header: invoice date + time on the right.
- **Bordered info box**: Invoice # | Patient name + healthcare ID + phone | Doctor name. Single bordered block with thin internal dividers, matching the PDF.
- **Line-items table** with columns in this exact order and alignment:
  - `Particulars` (left) ‚Äî uses `item.name || item.description`
  - `Unit` (center) ‚Äî `item.unit || "Nos"`
  - `Rate` (right) ‚Äî `item.unit_price`
  - `Qty` (center) ‚Äî `item.quantity`
  - `Amount` (right) ‚Äî `qty √ó unit_price`
  - All amounts right-aligned; clean borders, no gradients.
- **Totals block** bottom-right: Sub-Total, Discount (if any), GST (if any), then **Total Amount** wrapped in double horizontal rules (top + bottom border) matching the PDF.
- **Status pill** (Paid / Unpaid / Partial / Cancelled) next to Total.
- **Paid / Outstanding** lines below total when applicable.
- **Notes** block (if present), styled as a bordered footer note.
- **Footer**: centered clinic name + "Thank you for visiting" tagline, no "Generated by StethoScribe" branding (to match PDF footer).
- Use neutral colors (black/slate) instead of the teal gradient so Print output looks like the PDF.

`printInvoice()` and `escapeHtml()` are unchanged.

- `src/components/billing/PatientInvoicesTab.tsx` ‚Äî already calls `buildInvoiceHtml` / `downloadInvoicePdf` / `getInvoicePdfUrl`; no changes needed.
